#  Это прокси сервер для полученеия информации по городам
## Инструкция по запуску
1. Загрузить node modules
>> yarn install
>
>Или
>> npm install
2. Запустить сервер
>> yarn start
>
>Или
>> npm start

###Сервер будет доступен по 
> http://localhost:3001
* Получить json 
> http://localhost:3001/getJson

### Результатом является следующий json
> {
> 
> "data": []
> 
> }
